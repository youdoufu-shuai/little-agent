# Way - 我的个人智能助手

这是我自己写的一个 AI 小助手，名字叫 **Way**。

做这个项目的初衷很简单：我需要一个能真正帮我“干活”的 AI，而不只是陪聊。所以除了接了大模型（Gemini/DeepSeek）让它能聊天、能看图之外，我重点给它加了一些**实用的能力**，比如操作本地文件、连数据库查数据、生成图片等等。

它就像是个懂技术的私人管家，有事直接吩咐就行。

---

## 它的主要本事

*   **双脑架构（听着挺玄乎，其实就是好用）**
    *   **眼睛**：接了视觉模型，发截图、照片给它，它都能看懂。
    *   **脑子**：接了逻辑推理模型，写代码、分析问题比较靠谱。

*   **实干派工具箱**
    *   **文件操作**：能读取我电脑里的代码文件，帮我分析项目结构。
    *   **数据库小能手**：这是我最常用的功能。配好 MySQL 信息后，我想查什么数据直接告诉它（比如“帮我查下最近注册的用户”），它自己会写 SQL 去查并把结果告诉我，省得我还要开数据库客户端手敲 SQL。
    *   **绘画**：接了画图接口，能根据描述生成图片。

*   **百变人设**
    *   支持切换不同的人设。比如默认是“得力助手”，但我还加了个“墨宝”（古风少女），聊起天来文绉绉的，挺有意思。你也可以自己加喜欢的人设。

*   **界面**
    *   我自己手撸的 Web 界面，没用什么复杂的框架，主打一个轻量。
    *   支持深色/浅色模式切换（甚至图标都会跟着变）。
    *   有打字机效果，交互体验我觉得还行。

## 怎么跑起来

如果你也想在本地玩玩，按下面步骤来就行：

### 1. 准备环境
你需要有 Python 3.8 以上的环境。

### 2. 安装依赖
把项目下下来，然后在终端里运行：
```bash
pip install -r requirements.txt
```

### 3. 配置一下
在项目根目录下创建一个 `.env` 文件（参考 `.env.example`），把你的 API Key 填进去。
主要是这两个：
*   `VISION_API_KEY` (管看的)
*   `LOGIC_API_KEY` (管想的)

### 4. 启动
在终端运行：
```bash
uvicorn server:app --reload --host 127.0.0.1 --port 8000
```

然后打开浏览器访问 `http://127.0.0.1:8000` 就可以开始跟它聊天了。
