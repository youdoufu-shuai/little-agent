# 个人智能助手

这是一个由 **油豆腐** 和 **Fen ZhanWei** 开发的个人 AI 助手项目。

## 项目背景

没啥背景，想着可以让ai直接做出工作需要的成品文件，而不是还要我反复提需求然后做各种搬运工。

没有花里胡哨的功能，主打一个实用和轻量。

---

## 主要功能

*   **多模态交互**
    *   集成了视觉模型和逻辑推理模型，既能识别截图内容，也能处理复杂的逻辑分析和代码生成任务。

*   **本地文件操作**
    *   可以直接读取本地项目的文件和目录结构。
    *   场景举例：把项目目录发给它，让它快速梳理代码架构，或者读取某个报错的日志文件进行分析。

*   **数据库直连 (MySQL)**
    *   这是我们自己用的最多的功能。在界面上配置好数据库连接后，可以直接用自然语言查询数据。
    *   例如：输入“查一下最近注册的 5 个用户”，它会自动生成 SQL 并执行，直接把结果表格展示出来。

*   **人设管理**
    *   支持切换不同的 System Prompt。除了默认的助手模式，也预置了几个不同风格的人设（如古风风格等），也可以根据需要自己添加。

*   **轻量级 UI**
    *   前端使用原生 HTML/CSS/JS (Vanilla JS) 编写，没有引入复杂的框架，代码结构清晰，方便二次开发。
    *   支持深色/浅色主题切换。

## 快速开始

如果你对这个小工具感兴趣，可以按以下步骤在本地运行。

### 1. 环境准备
确保你的电脑上安装了 Python 3.8 或以上版本。

### 2. 安装依赖
克隆代码后，在项目根目录下运行：

```bash
pip install -r requirements.txt
```

### 3. 配置文件
复制 `.env.example` 为 `.env`，并填入必要的 API Key：

不在项目里填写也行，启动之后去 `http://127.0.0.1:8000` 配置一下。

```ini
# 视觉模型 API Key
VISION_API_KEY=your_key_here
# 逻辑模型 API Key
LOGIC_API_KEY=your_key_here
```

### 4. 启动服务
运行以下命令启动后端服务：

```bash
uvicorn server:app --reload --host 127.0.0.1 --port 8000
```

启动成功后，浏览器访问 `http://127.0.0.1:8000` 即可使用。
